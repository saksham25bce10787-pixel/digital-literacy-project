## Project Overview
This project has been created as part of the Digital Literacy course. The aim of this project is to understand how digital tools and platforms are used in real life. Through this project, I explored different areas such as creating presentations, building a professional profile, using coding platforms, understanding email etiquette, and learning about cyber safety.

## Task Summary

### Task 1 – Infographic
For this task, I created an infographic using Canva. It explains the concept of digital literacy, safe internet practices, and some useful tools for students.

### Task 2 – Digital Portfolio
In this task, I created and updated my profiles on platforms like GitHub, LinkedIn, and Kaggle. This helped me understand how to build a professional online presence.

### Task 3 – Platforms
I explored coding platforms and solved a beginner-level problem on HackerRank. I also created a Google Form quiz related to digital literacy.

### Task 4 – Email Etiquette
I wrote two professional emails and created a checklist for proper social media usage. This task helped me understand the importance of communication in a professional environment.

### Task 5 – Cybercrime Awareness
In this task, I studied phishing as a cybercrime and created a case study along with a prevention checklist to stay safe online.
